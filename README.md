# Pencatatan Sederhana Berbasis HTML, CSS dan JS
Aplikasi web sederhana untuk membuat, menampilkan, mengedit, dan menghapus catatan menggunakan HTML, CSS, dan JavaScript. Data disimpan di LocalStorage sehingga aplikasi dapat berjalan tanpa database dan tanpa server.

## Fitur
- Tambah catatan
- Edit catatan
- Hapus catatan
- Simpan data di LocalStorage

## Teknologi
- HTML
- CSS
- JavaScript
- LocalStorage

## Cara Menjalankan
Cukup buka file `index.html` di browser. Tidak memerlukan instalasi atau konfigurasi tambahan.

## Struktur Project
index.html
style.css
script.js

## Catatan
Project ini dibuat untuk memenuhi tugas praktikum Pemrograman Web dan sebagai latihan implementasi CRUD sederhana.
